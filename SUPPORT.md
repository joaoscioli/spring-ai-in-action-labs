# Support

This repository is maintained as a Spring AI engineering lab. Questions and
suggestions are welcome when they improve the safety, usefulness, or clarity of
the AI-assisted examples.

## Best Ways To Ask For Help

- Open an issue for unclear prompts, missing safety boundaries, API behavior, or
  documentation gaps.
- Include the affected lab, endpoint, prompt, or test when possible.
- Describe the expected model-assisted behavior and the observed result.

## Good Issue Examples

- A prompt does not explain its constraints clearly.
- An API response needs better validation or error handling.
- A test should cover unsafe, ambiguous, or malformed input.
- A lab should document where AI is useful and where deterministic code wins.

## Out Of Scope

- Production AI system support.
- Real API keys, private prompts, personal data, or customer data.
- Vendor-specific troubleshooting unrelated to the portfolio lab.
